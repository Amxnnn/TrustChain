# Project Name: TrustChain

TrustChain is a decentralized application (DApp) that revolutionizes supply chain transparency using the Ethereum Blockchain. By recording every product lifecycle event—from manufacturing to delivery—on an immutable ledger, TrustChain eliminates counterfeiting and builds trust between producers and consumers. With a stunning 3D interactive interface and IPFS-backed data storage, it brings the future of logistics to life today.
